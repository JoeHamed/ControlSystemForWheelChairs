# ControlSystemForWheelChairs
- AUGMENTED MULTIMODE CONTROL SYSTEM FOR WHEELCHAIRS IN HELP FOR THE NEEDY
- This project presents the development and evaluation of a multi-modal wheelchair control system designed to enhance user mobility and independence. The system offers a variety of control methods, including traditional joystick control, hands-free gaze control using eye movements and blinks, and person tracking that guides the wheelchair based on a designated companion's movements. The project prioritizes user safety by incorporating features like interrupt signals for obstacle detection and location sharing in person tracking mode. Additionally, a user-friendly GUI interface allows users to seamlessly switch between control methods based on their needs and preferences. This project details the existing system studies in this field, our system design, implementation, and results obtained from testing and evaluation. The project demonstrates the feasibility of a versatile control system that empowers users with diverse navigation options while prioritizing safety and ease of use. Future work directions explore further advancements in areas like battery life optimization, remote monitoring capabilities, and integrated health monitoring for enhanced user well-being.
